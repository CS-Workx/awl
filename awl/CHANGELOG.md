# Changelog

All notable changes to AWL Scanner will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-01-30

### Added
- Initial release
- PWA support for iOS
- Gemini Vision AI-powered OCR
- Automatic CSV and PDF generation
- Email delivery with attachments
- Contact management
- Service Worker for offline capability
- Mobile-first responsive design

### Features
- Scan attendance lists via camera
- AI extraction of participant data
- Digital CSV export
- PDF generation from scanned images
- Multi-recipient email sending
- Installable as home screen app

### Technical
- Node.js/Express backend
- Sharp for image processing
- PDF-lib for PDF generation
- Nodemailer for email
- Google Generative AI integration
